Code[modifier | modifier le code]
CAT est un code qui peut signifier :
l'île Cat aux Bahamas, selon la liste des codes AITA des aéroports.
cat est un code qui peut signifier :
catalan, selon les normes ISO 639-2 et 639-3 ;
.cat, l'extension de nom de fichier utilisée pour transmettre des programmes à une calculatrice Casio à l'aide du programme FA-123 ;
.cat, l'extension de nom de fichier d'un Security Catalog file, un format de fichier peu documenté de Microsoft destiné à assurer l'intégrité d'une liste de fichiers ;
cat, une commande Unix (le nom est en rapport avec le verbe anglais concatenate).

cellule anti-terroriste, dans les journées successives du feuilleton télévisé 24 heures chrono ;
conduite à tenir, une prise en charge devant une pathologie en médecine ;
certificat d'aptitude technique, un certificat nécessaire pour le passage au statut de gendarme de carrière ;
compte à terme, un produit financier (voir https://www.service-public.fr/particuliers/vosdroits/F2372) ;
chloramphénicol acétyltransférase, une enzyme intervenant dans la détoxication de l'antibiotique Chloramphénicol chez certaines bactéries ;
Centre d'aide par le travail ;
Centre d'analyse du terrorisme, le premier think tank européen d'analyse du terrorisme, fondé en 2014 ;
Commissariat de l'armée de terre, un service militaire ;
Commission des accidents du travail, un ancien organisme québécois ;
Compagnie Air Transport (de), une compagnie aérienne française ;
Compagnie d'affrètement et de transport, une ex-filiale du Groupe Renault spécialisée notamment dans le transport de véhicules ;
Compagnie d'assurances transports, une compagnie d'assurances marocaine ;
Confédération autonome du travail, une centrale syndicale française ;
Communauté d'agglomération Troyenne (aujourd'hui le Grand Troyes), regroupant Troyes et les communes voisines ;
en anglais
Central Africa Time (en français, heure d'Afrique centrale) ;
Computer-assisted translation (en français, traduction assistée par ordinateur) ;
Computer Aided Transceiver, un dispositif permettant de commander un émetteur-récepteur radio à l'aide d'un ordinateur ;
Civil Air Transport, une compagnie aérienne ;
Čerenkov Array at Thémis, un télescope à imagerie Tcherenkov atmosphérique installé sur le site de la centrale solaire Thémis entre 1996 et 2000.

Nom[modifier | modifier le code]
Cat est un mot qui peut signifier :
Cat, un langage de programmation informatique ;
Cat (ou Chat dans certaines traductions), un héros créé par Joan D. Vinge, voir Cycle de Cat ;
Cat , un jeu de rôles de John Wick.

cat, l'automutilation et la scarification ;
chat, en anglais ;
chat, en normand (le mot désigne aussi affectueusement les léopards du blason).
